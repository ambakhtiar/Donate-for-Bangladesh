What is the Document Object Model (DOM)?
The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of a document as a hierarchical tree of objects, allowing programmers to access, modify, and interact with the document's content, structure, and style. Think of it as a way to dynamically change or interact with the content and structure of a web page.
How do you select an element from the DOM?
You can select DOM elements using methods like getElementById, querySelector, or getElementsByClassName. Event delegation lets you handle events at a parent level using bubbling, reducing multiple listeners. Use .setAttribute, .style.property, or class methods like .classList.add() to change attributes and styles.
What is event delegation in the context of the DOM, and why is it useful?
Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events for its child elements, using event bubbling.
